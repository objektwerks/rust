Rust
----
>Rust feature tests, to include:
1. chrono
2. futures
3. hyper
4. reqwest
5. tokio

Build
-----
1. cargo clean
2. cargo build

Test
----
1. cargo test

Run
---
1. cargo run

Release
-------
1. cargo build --release ( see target/release/rust executable file )
2. ./target/release/rust

Executable
----------
1. rustc src/main.rs ( see ./main executable file )
2. ./main
